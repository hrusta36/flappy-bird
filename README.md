# Flappy Bird

A classic Flappy Bird clone built with vanilla JavaScript and the HTML5 Canvas API.

## Description
Guide your bird through the pipes by tapping the spacebar. 
Each pipe you pass earns you a point.

## Built With
- HTML5 Canvas API
- Vanilla JavaScript
- CSS3

## How to Run
1. Clone or download the project
2. Open the folder in VS Code
3. Install the Live Server extension if you haven't already
4. Right-click `flappybird.html` and select **Open with Live Server**
5. The game opens automatically in your browser

## Controls
| Key | Action |
|-----|--------|
| Space | Jump |
| Arrow Up | Jump |
| X | Jump |

## How to Play
- Press Space to start and jump
- Avoid the pipes
- Every pipe passed = 1 point
- Every 10 points you hear a sound
- Hit a pipe or fall = Game Over
- Press Space after Game Over to restart

## Requirements
- VS Code
- Live Server extension
- A modern browser (Chrome, Firefox, Edge)
